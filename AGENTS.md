# State Management Rules

- Prefer Zustand over Redux
- Never store server cache in Zustand
- Use selectors
- Avoid full-store subscriptions
- Use shallow selectors for grouped state

# Communication Rules

- Do not add preamble.
- Do not summarize what you are about to do.
- Do not end with 'let me know if you need anything else' or similar phrasing.
- Start your response with an answer directly.
